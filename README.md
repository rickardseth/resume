# Build

This repository loosely follows [Scripts To Rule Them
All](https://github.com/github/scripts-to-rule-them-all).

Download and install dependencies:

```
./script/bootstrap
```

Build the resume:

```
./script/build
```

View the output at `resume.pdf`.
